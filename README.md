#### It's an AngularJS version of [AlexKVal/react-thinking](https://github.com/AlexKVal/react-thinking/)

Frameworks used:
- Bootstrap (CSS only)
- AngularJS

In console run:

    $ git clone https://github.com/AlexKVal/products-bs3
    $ cd products-bs3
    $ bower install
    $ python -m SimpleHTTPServer 8000 & open http://localhost:8000
